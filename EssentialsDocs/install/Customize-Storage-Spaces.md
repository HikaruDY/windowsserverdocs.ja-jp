# <a name="customize-storage-spaces"></a>記憶域をカスタマイズします。

>Windows Server 2016 Essentials、Windows Server 2012 R2 Essentials での Windows Server 2012 Essentials を適用対象:

記憶域スペースを事前設定、ユーザー エクスペリエンス向上のため、PostIC.cmd から記憶域構成スクリプトを実行できます。 記憶域構成スクリプトは、Windows Server Essentials のインストールが完了したら後に、対象のコンピューターで実行する必要があります。
