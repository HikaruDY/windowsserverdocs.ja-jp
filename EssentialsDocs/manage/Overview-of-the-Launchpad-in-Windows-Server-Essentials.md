# <a name="overview-of-the-launchpad-in-windows-server-essentials"></a>Windows Server Essentials のスタート パッドの概要

>Windows Server 2016 Essentials、Windows Server 2012 R2 Essentials での Windows Server 2012 Essentials を適用対象:

Windows Server Essentials スタート パッドは、サーバーに接続するコンピューターが初めて時間にインストールされているサイズの小さいアプリケーションです。 スタート パッドは、認証されたユーザーにコンピューターのバックアップ、共有ファイルおよびメディア、リモート Web アクセス サイトなど、Windows Server Essentials の主な機能へのアクセスを提供します。 ユーザーは、ドメインに参加しているコンピューターと非ドメインに参加しているコンピューターのいずれかからこれらの機能にアクセスできます。 スタート パッドでは、リアルタイムの情報と、コンピューターの正常性についての通知も提供します。 管理者は、コンピューターがネットワークに接続されていない場合でも、サーバー ダッシュ ボードへのアクセス、スタート パッドを使用できます。  
  
 Oem および Windows Server Essentials のアドインを開発した独立系ソフトウェア ベンダー (Isv) は、スタート パッドを使用して、ネットワーク上のコンピューターにアドイン機能を拡張します。  
  
 次の Windows オペレーティング システムでは、Windows Server Essentials スタート パッドの使用をサポートします。  
  
-   **Windows 8**: すべてのエディション。  
  
-   **Windows 7**: すべてのエディション。  
-   **Windows 10**: すべてのエディション。 
  
 次のオペレーティング システムは、Windows Server Essentials スタート パッドの使用をサポートしていません。  
  
-   **追加のサーバー**: Windows Server オペレーティング システムを実行する追加のコンピューターで Windows Server Essentials スタート パッドを実行することはできません。
