##  <a name="BKMK_ConnectServer"></a>ローカル ネットワークからサーバーに接続する方法  
 Windows Phone 用 Windows Server Essentials、Windows 8、および Windows 8.1 の My Server アプリを正常に使用するには、まず、デバイスで、サーバー証明書をインストールする必要があります。 これにより、デバイスが、ローカル ネットワーク上の Windows Server Essentials を実行しているサーバーに接続します。 ローカル ネットワーク上のサーバーに接続して、Windows Phone または Windows 8 または Windows 8.1 を実行しているコンピューターにサーバー証明書をインストールするには、次の手順を使用します。  
  
#### <a name="to-connect-to-your-server-from-your-windows-phone"></a>Windows Phone からサーバーに接続するには  
  
1.  Windows 8 または Windows 8.1 を実行している、Windows Phone では、Internet Explorer を開きます。  
  
2.  アドレス バーで、次のように入力します。 **http://<yourservername\>/connect/default.aspx?get=caroot.cer**、し、Enter キーを押します。  
  
3.  Caroot.cer 証明書をインストールするメッセージが表示されたら、] をクリックして**インストール**します。  
  
4.  証明書のインストールを完了するまで待ちます。  
  
    > [!NOTE]
    >  証明書のインストールが完了したらにサインインする My Server アプリを Windows Phone のサーバー名とネットワーク資格情報を使用しています。  
  
#### <a name="to-connect-to-your-server-in-your-local-network-from-a-computer-running-windows-8-or-windows-81"></a>Windows 8 または Windows 8.1 を実行しているコンピューターからローカル ネットワーク内のサーバーに接続するには  
  
1.  Windows 8 または Windows 8.1 を実行しているコンピューターでは、Internet Explorer を開きます。  
  
2.  アドレス バーで、次のように入力します。 **http://<yourservername\>/connect/default.aspx?get=caroot.cer**、し、Enter キーを押します。  
  
3.  Caroot.cer 証明書をインストールするメッセージが表示されたら、] をクリックして**開く**します。  
  
4.  **証明書**ダイアログ ボックスで、をクリックして**証明書のインストール**します。  
  
5.  証明書のインポート ウィザードで選択**ローカル マシン**、ストアの場所として。  
  
6.  選択**すべての証明書を次のストアに配置**を参照して、**信頼されたルート証明機関**場所。  
  
7.  ウィザードを完了する指示に従います。  
  
    > [!NOTE]
    >  証明書のインストールが完了したらにサインインする My Server アプリを Windows 8 または Windows 8.1 用、サーバー名とネットワーク資格情報を使用しています。
