# ライセンス認証エラー コードのトラブルシューティング

**ホーム ユーザー**この記事では、サポート担当者および IT プロフェッショナルによる使用の想定しています。 Windows のライセンス認証エラー メッセージに関する詳細情報を探している場合は、次の Microsoft web サイトに移動します。

[Windows のライセンス認証エラーに関するヘルプ](https://support.microsoft.com/help/10738/windows-10-get-help-with-activation-errors)
