# <a name="recommended-settings-for-vdi-desktops"></a>VDI デスクトップで推奨される設定

>適用先:Windows Server 2016、windows 10 の Windows Server (半期チャネル)

Microsoft Desktop Virtualization は、デバイスの構成とユーザーをして実行するより早く会社のアプリケーションとデスクトップのインスタント設定を有効にするとネットワーク条件に自動的に検出し、そのレガシへのアクセスを提供する ITWindows 10 への移行中のアプリケーション。

Windows 10 オペレーティング システムがすぐチューニングは非常にうまく調整するための機会は具体的には、企業の Microsoft 仮想デスクトップ インフラストラクチャ (VDI) 環境について詳細にします。 VDI 環境で、最初から多くのバック グラウンド サービスとタスクは無効になります。

このトピックが、ブルー プリントではなく、ガイドか開始点です。 推奨事項には、ため、実際のシナリオで、特定の設定を調整することの利点とコストを考慮する必要がありますを使用して、希望の機能が無効にする可能性があります。

これらの手順と推奨設定は、Windows 10 1607 (バージョン 10.0.1393) に関連するされます。

> [!NOTE]  
> このトピックで特に記載しているすべての設定を残してかまいません VDI 機能ほど影響を与えずに、既定値 (または、要件やポリシーごとにセット)。

## <a name="microsoft-store-apps"></a>Microsoft Store アプリ
VDI イメージ; から削除する Microsoft Store アプリがいくつか削除しても、CPU 使用率を減らすし、ディスク領域を節約します。 削除の候補は次のとおりです。

- Get Office
- Skype (プレビュー)
- (特にインターネット接続がない) 場合に開始します。
- フィードバック Hub
- ソリティア コレクション
- 有料の Wi-fi プロファイルおよび携帯電話

VDI のイメージを作成するために使用される既定のユーザー プロファイルをカスタマイズするには、ビルトイン Administrator アカウントを使用します。 既に有効にしない場合は、ローカル ユーザーとグループをコンピューターの管理を使用して行います。 次の手順を完了する管理者アカウントにログインします。

> [!NOTE]  
> システム アプリなどのストア アプリを削除しません。 これらは、再インストール困難です。 他のアプリは、ストアから簡単に reinstallable です。

### <a name="delete-unwanted-apps-from-the-administrator-user-profile"></a>管理者のユーザー プロファイルから望ましくないアプリを削除します。
1. Windows PowerShell で実行`Get-AppxPackage | ft PackageFamilyName`にインストールされているアプリの一覧を参照してください。
2. アンインストールする各アプリ パッケージ ツールは、この形式の例のコマンドレットを実行します。

    `Get-AppxPackage *messaging* | Remove-AppxPackage`

    `Get-AppxPackage *WindowsMaps* | Remove-AppxPackage`

    `Get-AppxPackage *ZuneMusic* | Remove-AppxPackage`

### <a name="delete-the-payload-of-unwanted-store-apps"></a>不要なストア アプリのペイロードを削除します。
これにより、アプリで再インストールされるできなくなります。
1. 一覧のストア アプリと、このコマンドレットでストレージにデータを準備しているその他の項目:`Get-AppxProvisionedPackage -Online`します。
2. 指定されたパッケージを削除する`Remove-AppxProvisionedPackage -Online -PackageName MyAppPackage`、手順 1. から返される適切な MyAppPackage を使用します。 たとえば、Zune に関連するパッケージを削除する場合は実行`Remove-AppxProvisionedPackage -Online -PackageName Microsoft.ZuneMusic_2019.17012.10311.0_neutral_~_8wekyb3d8bbwe`します。

## <a name="removing-other-items"></a>その他の項目を削除します。
OneDrive のアイコンとアプリを削除、システムのアイコンをオフにする、およびダウンロードされた更新プログラムを削除できます。

### <a name="remove-onedrive-icon-and-app"></a>OneDrive のアイコンとアプリを削除します。
1. クリックして**開始**までスクロールし、 **OneDrive**アイコン。
2. 右クリックし、 **OneDrive**アイコンをポイントして**詳細**、 をクリックし、**ファイルの場所を開く**します。
3. 右クリックし、 **OneDrive**そのファイルの場所をクリックします アイコン**削除**します。

OneDrive アプリを削除するには。
1. クリックして**開始**までスクロールし、 **OneDrive**アイコン。
2. 右クリックし、 **OneDrive**アイコン、およびクリック**アンインストール**します。 プログラムと機能が表示されます。
3. プログラムと機能 で右クリック**Microsoft OneDrive**クリック**アンインストール**します。

### <a name="programs-and-features-from-previous-versions-of-control-panel"></a>プログラムと機能 (からコントロール パネルの以前のバージョン)
1. プッシュ、**開始**ボタンに「*コントロール*、し、ENTER キーを押します。
2. タップするか、ダブルクリックして**プログラムと機能**します。
3. 一番左にある **コントロール パネル ホーム**をタップまたはクリックして**オンまたはオフにする Windows 機能**します。 新しいユーザー インターフェイスが開きます。
4. しますまたは、たとえば、基本イメージに必要なしないすべての項目のチェック ボックスをオフにします。**SMB 1.0/CIFS ファイル共有のサポート**します。

### <a name="turn-system-icons-off"></a>システムのアイコンをオフにします。
1. をクリックしてまたはプッシュ**開始**、順にクリックします**設定**(歯車アイコン)。
2. **設定を見つける**テキスト領域で、「*タスク バー*順にクリックします**タスク バー設定**します。
3. で、**タスク バー**セクションで、スクロールするか、下へスワイプ、**通知領域**セクション。
4. をクリックしてまたはタップします**システムのアイコンをオンまたはオフに**、し、システムの各アイコン オンまたはオフに、イメージに必要であればとします。

### <a name="delete-downloaded-updates"></a>ダウンロードした更新プログラムを削除します。
1. ファイル エクスプ ローラーを使用してに移動**C:\Windows\Software Distribution\Download**します。
2. すべてのファイルとそのディレクトリ内のフォルダーを削除します。
