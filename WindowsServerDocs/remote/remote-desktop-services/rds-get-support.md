情報だけでなく、ここで、Windows Server 2016 ライブラリについて説明し、リモート デスクトップ サービスのヘルプを表示する、次のリソースを使用できます。

- 一般にリモート デスクトップ サービスに関するディスカッションに参加、 [RDS に関する TechNet フォーラム](https://aka.ms/technetforum-rds)します。
