# クライアント アプリを比較します。

>適用対象: Windows 10、Windows 8.1、Windows Server 2019、Windows Server 2016、Windows Server 2012 R2

多くの場合求めるメッセージがさまざまなリモート デスクトップ クライアント アプリのと比較します。 同じ操作を行うすべてかどうか。 これらの質問に対する回答を示します。

次の表では、デバイスとリモート デスクトップ接続アプリ、ユニバーサル アプリ、Android アプリ、iOS アプリ、macOS アプリと web クライアント上の他のリダイレクトのサポートを比較します。 これらの表では、リモート セッションで 1 回アクセスできるようにするリダイレクトについて説明します。 

場合は、個人をデスクトップにリモートでは、セッションの**その他の設定**で構成できる追加のリダイレクトします。 場合は、リモート デスクトップ アプリやアプリは、組織で管理は、管理者が有効またはグループ ポリシー設定でのリダイレクトを無効にできます。
