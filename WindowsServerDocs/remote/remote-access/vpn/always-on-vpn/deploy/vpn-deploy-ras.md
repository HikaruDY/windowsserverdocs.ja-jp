# 手順 3.  Always On VPN 用にリモート アクセス サーバーを構成する

>適用対象: Windows Server (半期チャネル)、Windows Server 2016、Windows Server 2012 R2、Windows 10

& #171 です。 [**前:** 手順 2 です。サーバー インフラストラクチャを構成します。](vpn-deploy-server-infrastructure.md)<br>
& #187 です。 [**前:** 手順 4 です。インストールし、構成のネットワーク ポリシー サーバー (NPS)](vpn-deploy-nps.md)


RRAS はさまざまな機能をサポートしているため、ルーターとリモート アクセス サーバーの両方を実行する設計されています。 この展開の目的上、これらの機能の小さなサブセットのみが必要な: IKEv2 VPN 接続および LAN ルーティングのサポート。

IKEv2 は、VPN トンネリング プロトコルのコメント 7296 インターネット技術標準化委員会要求で説明されています。 IKEv2 の主な利点は、基になるネットワーク接続の中断を許容できることができます。 たとえば、接続が一時的に失われた場合、または、ユーザーでは、1 つのネットワークからクライアント コンピューターを移動する場合は、IKEv2 自動的に接続が復元される VPN ネットワーク接続を再び-ユーザーの介入なしすべてします。

あり、サーバーのセキュリティの使用量を抑えることが使用されていないプロトコルを無効にすると、IKEv2 接続をサポートするために RRAS サーバーを構成します。 さらに、VPN クライアントに静的アドレス プールからアドレスを割り当てるサーバーを構成します。 プールまたは DHCP サーバーから現実的アドレスを割り当てることができます。ただし、DHCP サーバーを使用して、設計複雑さを追加し、最小限のメリットを提供します。


>[!Important]
>これが重要です。
>- 物理サーバーでは、2 つのイーサネット ネットワーク アダプターをインストールします。 VM に VPN サーバーをインストールする場合は 2 外部仮想スイッチ、各物理ネットワーク アダプターのいずれかを作成する必要があります。VM の 1 つの仮想スイッチに接続されている各ネットワーク アダプターと 2 つの仮想ネットワーク アダプターを作成します。
>
>- 外部の境界ネットワークに接続されている 1 つのネットワーク アダプターと境界領域の内部ネットワークに接続されている 1 つのネットワーク アダプターがエッジと、内部のファイアウォールの間で、境界ネットワーク上のサーバーをインストールします。


>[!Warning]
>開始する前に、VPN サーバーで IPv6 を有効にすることを確認します。 それ以外の場合、接続を確立できないされ、エラー メッセージが表示されます。

## リモート アクセスを VPN サーバーとして構成します。

このセクションでは、他の VPN プロトコルからの接続を拒否、承認された VPN クライアントが接続している IP アドレスの発行の静的 IP アドレス プールを割り当てる IKEv2 VPN 接続を許可するリモート アクセス VPN を構成できます。

1.  VPN サーバーで、サーバー マネージャーで、**通知**フラグをクリックします。

2.  **タスク**のメニューで、**取得、開始ウィザードを開く**をクリックします。<p>リモート アクセスの構成ウィザードが開きます。 

    >[!NOTE] 
    >リモート アクセスの構成ウィザードは、サーバー マネージャーの背後にあるを開く可能性があります。 ウィザード時間がかかりすぎるを開く場合は、移動またはウィザードが背後にあるかどうかを確認するサーバー マネージャーを最小限に抑えます。 必要でない場合は、ウィザードを初期化するまで待ちます。

3.  **VPN のみの展開**をクリックします。<p>ルーティングとリモート アクセスの Microsoft 管理コンソール (MMC) が開きます。

4.  VPN サーバーを右クリックし、**構成を有効にするルーティングとリモート アクセスと**をクリックします。<p>ルーティングとリモート アクセス サーバーのセットアップ ウィザードが開きます。

5.  ルーティングとリモート アクセス サーバーのセットアップ ウィザードへようこそ]、 **[次へ]** をクリックします。

6.  **構成**では、**カスタム構成**をクリックし、 **[次へ]** をクリックします。

7.  **カスタム構成**では、 **VPN アクセス**をクリックし、 **[次へ]** をクリックします。<p>完了ルーティングとリモート アクセス サーバーのセットアップ ウィザードが開きます。

8.  **完了**ウィザードを閉じる] をクリックして、ルーティングとリモート アクセスのダイアログ ボックスを閉じるには、 **[ok]** をクリックします。
