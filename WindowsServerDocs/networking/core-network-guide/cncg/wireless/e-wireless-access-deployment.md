### <a name="bkmk_wirelessaps"></a>ワイヤレス Ap を構成します。

#### <a name="to-configure-your-wireless-aps"></a>ワイヤレス Ap を構成するには  

- **SSID**します。 ワイヤレス network\(s\) の名前を指定 \ (たとえば、ExampleWLAN\)。 これがワイヤレス クライアントに提供される名前です。

- **暗号化**します。 WPA2\ エンタープライズ \(preferred\) または WPA\-エンタープライズと AES \(preferred\) または TKIP 暗号化の暗号で、ワイヤレス クライアント コンピューターのネットワーク アダプターによってサポートされているバージョンに応じてを指定します。

- **ワイヤレス AP の IP アドレス \(static\)**します。 各 AP では、サブネットの DHCP スコープの除外範囲内に一意の静的 IP アドレスを構成します。 DHCP によって割り当てから除外されているアドレスを使用して、DHCP サーバーは、コンピューターやその他のデバイスに同じ IP アドレスを割り当てるできなくなります。

- **サブネット マスク**します。 このワイヤレス AP を接続する LAN のサブネット マスクの設定と一致するを構成します。  

- **DNS 名**します。 一部のワイヤレス Ap は、DNS 名で構成できます。 ネットワーク上の DNS サービスは、IP アドレスに DNS 名を解決することができます。 この機能をサポートする各ワイヤレス AP 上の DNS 解決の一意の名前を入力します。  

- **DHCP サービス**します。 組み込みの DHCP サービス、ワイヤレス AP の場合は、無効にします。  

- **RADIUS の共有シークレット**します。 Ap のグループ別の NPS の RADIUS クライアントとして構成を計画している場合を除き、ワイヤレス AP ごとに一意の RADIUS 共有シークレットを使用します。 NPS で Ap グループを構成する場合は、共有シークレットをグループのすべてのメンバーと同じにする必要があります。 さらに、各共有シークレットを使用するには、少なくとも 22 文字の大文字が混在していると小文字の英字、数字、および句読点のランダムな順序をする必要があります。 ランダム性を確実には、NPS で見つかったランダムな文字ジェネレーターなどのランダムな文字ジェネレーターを使用することができます**構成 802.1 X**ウィザードで、共有シークレットを作成します。

>[!TIP]
>各ワイヤレス AP の共有シークレットを記録し、安全なオフィスなどのセキュリティで保護された場所に保管します。 NPS の RADIUS クライアントを構成するときに、各ワイヤレス AP の共有シークレットを知る必要があります。  

- **RADIUS サーバーの IP アドレス**します。 NPS を実行しているサーバーの IP アドレスを入力します。

- **UDP port\(s\)**します。 既定では、NPS は、認証メッセージおよび 1813 および 1646 アカウンティング メッセージ用の UDP ポート 1812、1645、UDP ポートを使用します。 Ap でこれら同じ UDP ポートを使用は、別のポートを使用する正当な理由がある場合は Ap を新しいポート番号で構成するだけでなく、また Ap と同じポート番号を使用するように NPS サーバーのすべてを再構成することを確認することをお勧めします。 Ap と NPS サーバーを同じ UDP ポートを構成しなかった場合 NPS ことはできませんが表示される、Ap からの接続要求の処理や、ネットワーク上のすべてのワイヤレス接続試行は失敗します。

- **Vsa**します。 一部のワイヤレス Ap では、vendor\ 固有の属性 \(VSAs\) 完全ワイヤレス AP の機能を提供する必要があります。 Vsa は、NPS のネットワーク ポリシーに追加されます。

- **DHCP フィルター**します。 クライアントをブロックするワイヤレスから、ネットワークへの UDP ポート 68 から IP パケットを送信するように、ワイヤレス AP 製造元で説明されているワイヤレス Ap を構成します。

- **DNS のフィルタ リング**します。 クライアントをブロックするワイヤレス ネットワークに TCP または UDP ポート 53 から IP パケットを送信できないように、ワイヤレス AP 製造元で説明されているワイヤレス Ap を構成します。
