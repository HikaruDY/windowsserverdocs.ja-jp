# <a name="bitsadmin-cache-and-getexpirationtime"></a>bitsadmin キャッシュと getexpirationtime



キャッシュの有効期限を取得します。

## <a name="syntax"></a>構文

```
bitsadmin /Cache /GetExpirationTime 
```

## <a name="BKMK_examples"></a>例

次に、キャッシュの有効期限を取得します。
```
C:\>bitsadmin /Cache /GetExpirationTime
```
