# <a name="bitsadmin-util-and-help"></a>bitsadmin util とヘルプ



コマンドラインの使用法を表示、 **Util**スイッチ。

## <a name="syntax"></a>構文

```
Bitsadmin /Util /Help 
```

## <a name="BKMK_examples"></a>例

次のコマンドのコマンド ライン ヘルプを表示する、 **Util**スイッチ。
```
Bitsadmin /Util /Help
```
