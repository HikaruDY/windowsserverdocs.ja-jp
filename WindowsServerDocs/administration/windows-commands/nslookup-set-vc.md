# <a name="nslookup-set-vc"></a>nslookup set vc



またはを使用して、サーバーへの要求を送信するときに、仮想回線を使用しないことを指定します。

## <a name="syntax"></a>構文

```
set [no]vc
```

## <a name="parameters"></a>パラメーター

| パラメーター |                                              説明                                               |
|-----------|--------------------------------------------------------------------------------------------------------|
| **novc**  | サーバーに要求を送信するときは、仮想回線を使用しないを指定します。 既定値は**novc**します。 |
|  **vc**   |             サーバーに要求を送信するときに常に仮想回線を使用することを指定します。             |
|   {0} のヘルプ   |                                                   ?}                                                   |
