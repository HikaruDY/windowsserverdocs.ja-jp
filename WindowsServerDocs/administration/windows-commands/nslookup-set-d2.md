# <a name="nslookup-set-d2"></a>nslookup set d2



完全なデバッグ モードを有効または無効にします。 すべてのパケットのすべてのフィールドが出力されます。

## <a name="syntax"></a>構文

```
set [no]d2
```

## <a name="parameters"></a>パラメーター

| パラメーター |                             説明                              |
|-----------|----------------------------------------------------------------------|
| **nod2**  | 完全なデバッグ モードをオフにします。 既定の構文は**nod2**します。 |
|  **d2**   |                 完全なデバッグ モードをオンにします。                  |
|   {0} のヘルプ   |                                  ?}                                  |
