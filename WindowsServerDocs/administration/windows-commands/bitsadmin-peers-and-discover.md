# <a name="bitsadmin-peers-and-discover"></a>bitsadmin ピアし、検出



もう一度ピアを検出します。

## <a name="syntax"></a>構文

```
bitsadmin /Peers /Discover 
```

## <a name="BKMK_examples"></a>例

次の例では、ピアを検出します。
```
C:\>bitsadmin /Peers /Discover
```
