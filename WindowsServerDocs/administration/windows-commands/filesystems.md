# <a name="filesystems"></a>ファイル システム



フォーカスがあるボリュームの現在のファイル システムに関する情報が表示され、ボリュームのフォーマットがサポートされているファイル システムの一覧します。

## <a name="syntax"></a>構文

```
filesystems
```

## <a name="remarks"></a>注釈

-   この操作を成功させるのには、ボリュームを選択してください。 使用して、 **ボリュームを選択して** コマンドのボリュームを選択し、それにフォーカスをします。
