# <a name="bitsadmin-cache-and-getlimit"></a>bitsadmin キャッシュと getlimit



キャッシュの制限値を取得します。

## <a name="syntax"></a>構文

```
bitsadmin /Cache /GetLimit 
```

## <a name="BKMK_examples"></a>例

次の例では、キャッシュの制限値を取得します.
```
C:\>bitsadmin /Cache /GetLimit 
```
