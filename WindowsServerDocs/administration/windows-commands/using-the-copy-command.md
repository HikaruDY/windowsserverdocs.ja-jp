>適用先:Windows Server (半期チャネル)、Windows Server 2016、Windows Server 2012 R2、Windows Server 2012

イメージ、ドライバー グループをコピーします。

|[コピー イメージのコマンドを使用します。](using-the-copy-image-command.md)|同じイメージ グループ内にあるイメージをコピーします。|
