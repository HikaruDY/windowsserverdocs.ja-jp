# <a name="nslookup-set-all"></a>nslookup set all



構成設定の現在の値を出力します。

## <a name="syntax"></a>構文

```
set all 
```

## <a name="parameters"></a>パラメーター

|パラメーター|説明|
|---------|-----------|
|{0} のヘルプ | ?}|簡単な概要を表示します。 **nslookup**サブコマンドします。|

## <a name="remarks"></a>注釈

-   **すべての設定**も、既定のサーバーとコンピューター (ホスト) に関する情報を出力します。
