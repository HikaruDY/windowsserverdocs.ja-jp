# <a name="telnet-status"></a>telnet: ステータス

>適用先:Windows Server (半期チャネル)、Windows Server 2016、Windows Server 2012 R2、Windows Server 2012

ステータス情報を表示します。   
## <a name="syntax"></a>構文  
```  
st[atus]  
```  
### <a name="parameters"></a>パラメーター  
なし  
## <a name="BKMK_Examples"></a>例  
状態情報が表示されます。  
```  
st  
```
