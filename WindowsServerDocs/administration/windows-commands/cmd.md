# <a name="cmd"></a>Cmd

コマンド インタープリター、Cmd.exe の新しいインスタンスを開始します。 パラメーターを指定せずに使用されている場合**cmd**オペレーティング システムのバージョンおよび著作権情報が表示されます。

## <a name="syntax"></a>構文

```
cmd [/c|/k] [/s] [/q] [/d] [/a|/u] [/t:{<B><F>|<F>}] [/e:{on|off}] [/f:{on|off}] [/v:{on|off}] [<String>]
```

## <a name="parameters"></a>パラメーター

|パラメーター|説明|
|---------|-----------|
|/c|指定されたコマンドを実行*文字列*停止します。|
|/k|指定されたコマンドを実行*文字列*を続行します。|
|/s|変更の処理を行う*文字列*後 **/c**または **/k**します。|
|/q|エコーの電源をオフにします。|
|/d|自動実行コマンドの実行を無効にします。|
|/a|内部コマンドの出力をパイプまたはファイルとして National Institute (ANSI 規格) を書式設定します。|
|/u|内部コマンドの出力をパイプまたはファイルを Unicode として書式設定します。|
|/t: {\<B\>\<F\>\|\<F\>}|バック グラウンドの設定 (*B*) と前景色 (*F*) の色。|
|/e:on|コマンド拡張機能を有効にします。|
|/e:off|コマンド拡張機能を無効にします。|
|/f:on|ファイルとディレクトリ名の補完機能を有効にします。|
|/f: オフ|ファイルとディレクトリ名の補完機能を無効にします。|
|/v:on|有効では、環境変数の拡張を遅延します。|
|/v:off|環境変数の拡張が遅延を無効にします。|
|\<文字列 >|実行するコマンドを指定します。|
|/?|コマンド プロンプトにヘルプを表示します。|
