# <a name="end-backup"></a>末尾のバックアップ




完全バックアップのセッションとの問題を終了、 **BackupComplete**イベントが必要な場合は、適切なライターの状態。

## <a name="syntax"></a>構文

```
end backup
```

## <a name="remarks"></a>注釈

-   このコマンドは、すべての完全バックアップを使用する必要があります。
