# <a name="bitsadmin-cache-and-list"></a>bitsadmin キャッシュとリスト



すべてのキャッシュ エントリを一覧表示します。

## <a name="syntax"></a>構文

```
bitsadmin /Cache /List [/Verbose] 
```

## <a name="BKMK_examples"></a>例

次の例では、詳細な形式ですべてのキャッシュ エントリが一覧表示します。
```
C:\> bitsadmin /Cache /List /Verbose
```
