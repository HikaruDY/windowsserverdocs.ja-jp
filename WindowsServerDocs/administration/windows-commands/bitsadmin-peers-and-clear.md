# <a name="bitsadmin-peers-and-clear"></a>bitsadmin ピアと消去



ピアのリストをクリアします。

## <a name="syntax"></a>構文

```
bitsadmin /Peers /Clear 
```

## <a name="BKMK_examples"></a>例

次の例では、ピアのリストをクリアします。
```
C:\>bitsadmin /Peers /Clear
```
