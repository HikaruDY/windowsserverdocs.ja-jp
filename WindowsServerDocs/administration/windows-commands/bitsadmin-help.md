# <a name="bitsadmin-help"></a>bitsadmin help



コマンドラインの使用状況を表示します。

## <a name="syntax"></a>構文

```
bitsadmin /help | /?
```

## <a name="BKMK_examples"></a>例

次の例では、コマンド ライン ヘルプを取得します。
```
C:\>bitsadmin /help
```
