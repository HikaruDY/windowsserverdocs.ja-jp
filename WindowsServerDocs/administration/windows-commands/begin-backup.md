# <a name="begin-backup"></a>バックアップを開始します。



完全バックアップのセッションを開始します。

## <a name="syntax"></a>構文

```
begin backup
```

## <a name="remarks"></a>注釈

-   このコマンドは、既定のコピーのバックアップ設定をオーバーライドします。
