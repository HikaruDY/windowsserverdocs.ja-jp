# <a name="nslookup-set-port"></a>nslookup set port



指定された値には、既定の TCP または UDP ドメイン ネーム システム (DNS) 名サーバー ポートを変更します。

## <a name="syntax"></a>構文

```
set port=<Port>
```

## <a name="parameters"></a>パラメーター

| パラメーター |                                          説明                                          |
|-----------|-----------------------------------------------------------------------------------------------|
|  \<Port>  | 既定の TCP または UDP の DNS 名サーバー ポートの新しい値を指定します。 既定のポートは 53 です。 |
|   {0} のヘルプ   |                                              ?}                                               |
