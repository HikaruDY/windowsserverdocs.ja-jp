# <a name="cls"></a>cls



コマンド プロンプト ウィンドウをクリアします。

このコマンドを使用する方法の例については、[例](#BKMK_examples)を参照してください。

## <a name="syntax"></a>構文

```
cls
```

## <a name="parameters"></a>パラメーター

|パラメーター|説明|
|---------|-----------|
|/?|コマンド プロンプトにヘルプを表示します。|

## <a name="BKMK_examples"></a>例

空のウィンドウに戻って、コマンド プロンプト ウィンドウに表示されるすべての情報をクリアするには、次のように入力します。
```
cls 
```
