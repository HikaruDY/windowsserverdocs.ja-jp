# <a name="bitsadmin-cache-and-clear"></a>bitsadmin キャッシュと消去



ローカル キャッシュを削除します。

## <a name="syntax"></a>構文

```
bitsadmin /Cache /Clear 
```

## <a name="BKMK_examples"></a>例

次の例では、ローカル キャッシュを削除します。
```
C:\>bitsadmin /Cache /Clear 
```
