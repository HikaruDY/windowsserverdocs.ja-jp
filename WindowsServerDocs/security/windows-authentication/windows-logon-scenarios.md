## <a name="BKMK_NetworkLogon"></a>ネットワーク ログオン
ネットワーク ログオンは、ユーザー、サービス、またはコンピューターの認証が行われた後にのみ使用できます。 ネットワークのログオン時に、プロセスは、データを収集するのに資格情報の入力] ダイアログ ボックスを使用しません。 代わりに、資格情報を既に確立されて、または別の資格情報を収集する方法を使用します。 このプロセスは、ユーザーがアクセスしようとしている任意のネットワーク サービスへのユーザーの身元を確認します。 別の資格情報を提供する必要がない限り、このプロセスはユーザーに通常表示ではありません。

この種類の認証を提供するには、セキュリティ システムには、これらの認証メカニズムが含まれています。

-   Kerberos version 5 プロトコル

-   公開キー証明書

-   Secure Sockets Layer/トランスポート層セキュリティ (SSL/TLS)

-   Digest

-   NTLM, for compatibility with Microsoft Windows NT 4.0-based systems
